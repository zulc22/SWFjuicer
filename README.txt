Searches any file for valid SWF data barried within binary files.
This is not advanced enough software to search through any compressed data.
If you can't find something you're looking for, try running your files through 7-Zip.
Created for reverse-engineering of the Leapster 1 and 2, but can be
useful for many reverse-enginerring projects related to Flash.

SWFjuicer requires at least Python 3.6.
